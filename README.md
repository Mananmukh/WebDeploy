# Faculty-style GitHub Pages site (Bootstrap + Jekyll)

## Deploy
Upload the contents of this folder to your GitHub repo root. Then enable GitHub Pages:
Settings → Pages → Deploy from a branch → main + /(root)

## If your repo is NOT <username>.github.io
Edit `_config.yml` and set:
baseurl: "/<repo-name>"

Example: repo name is Website → baseurl: "/Website"

## Replace your photo
Replace `assets/profile.jpg` with your real photo (keep the same filename).
